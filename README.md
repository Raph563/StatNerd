# StatNerd

StatNerd is the analytics addon for Grocy.

## Dependency
StatNerd requires NerdCore:
- `Raph563/NerdCore`

Without NerdCore, StatNerd runtime is disabled by design.

## Runtime files
- StatNerd payload: `config/data/custom_js_nerdstats.html`
- Composed active file: `config/data/custom_js.html`

## Install order
1. Install NerdCore
2. Install StatNerd
3. (Optional) Install ProductHelper
